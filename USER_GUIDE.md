# Audio Spectrum Overlay Maker v1.1.2 (setup fix) User Guide

## Setup requirement

Use Python 3.10 or later. On Windows, `setup.bat` creates the local `.venv` with `py -3`, so confirm `py -3 --version` reports Python 3.10+ before setup.

Audio Spectrum Overlay Maker は、音源ファイルを解析し、音に同期して動くスペクトラムアナライザー風の MP4 オーバーレイ動画を生成するローカルアプリです。出力動画には音声は入りません。音源は解析にだけ使用します。

v1.1.1 では、動画編集ソフトで安定して重ねるための **比較合成用ペア出力** を中心機能として整理しました。

## 1. 基本の使い方

1. `start.bat` でアプリを起動します。
2. 「音源」で解析する音声ファイルを選びます。
3. 「出力先」で保存先フォルダを選びます。
4. プリセットを選びます。まずは `01 Dynamic Standard` が基本です。
5. 「見た目プレビュー」で形を確認します。
6. 「動きプレビュー」で実際の音源に対する動きを確認します。
7. 「30秒プレビュー動画生成」で短い素材を作って動画編集ソフトで確認します。
8. 問題なければ「全体生成」を行います。

## 2. 出力される動画

通常出力は、黒背景にスペアナバーを描いた無音 MP4 です。動画編集ソフトでは「比較（明）」「Lighten」「Screen」などで重ねる想定です。

ペア出力をオンにすると、通常のメイン動画に加えて、比較（暗）用のマット動画も出力します。

```text
メイン動画：黒背景 + 通常のスペアナ
マット動画：白背景 + 黒いスペアナ形状
```

## 3. 比較合成用ペア出力

下部の「全体生成」ボタンの横にある **比較合成用マットも出力** をオンにすると、30秒プレビュー生成と全体生成の両方でマット動画も出力されます。

動画編集ソフト側では、次の順番で重ねます。

```text
下：完成済み動画または背景動画
中：マット動画  → 比較（暗）
上：メイン動画  → 比較（明）
```

この方式では、まずマット動画でスペアナの形状部分だけを暗く抜き、その上にメイン動画のスペアナを乗せます。クロマキーよりもエッジが安定しやすく、色付きバーやグラデーションも扱いやすくなります。

## 4. 見た目タブ

見た目タブでは、主にスペアナの形を設定します。

- 表示タイプ：片側バー / 両側バー。
- 色モード：縦グラデーション / 帯域グラデーション。
- スクロール：なし / 左 / 右。
- シフト間隔：何フレームごとに帯域を1つ動かすか。
- バー本数：18 / 24 / 32 / 48 / 64。
- バー色1 / バー色2：2色を同じにすると実質単色になります。
- バー幅、角丸、最大高さ、左右余白、下余白。
- 横幅、高さ、fps。

背景色は通常触る必要がなくなったため、詳細設定タブに移動しています。標準の黒背景のまま使うのが基本です。ペア出力時のマット背景は自動的に白になります。

## 5. 生成・動きタブ

通常ユーザー向けの動き調整です。

- 反応の速さ：バーが音の変化にどれだけ速く追従するか。
- 上下動の強さ：ピーク後にどの程度落ちるか。
- 感度：全体的な反応感度。
- 動きの細かさ：内部解析バー数や動きの解像感。
- 高低差の出方：帯域ごとの差をどのくらい強く出すか。
- 動きプレビューに適した区間を自動検出。
- 動画プレビュー秒数。

## 6. 詳細設定タブ

上級者向けの内部パラメータです。通常は変更しなくて構いません。

背景色もここにあります。これは特殊用途・検証用の設定です。通常の素材出力では黒背景、ペア出力のマットでは自動的に白背景を使います。

## 7. ファイル名と上書き防止

既存ファイルは上書きしません。通常は ` (1)`, ` (2)` のように番号を付けます。

ペア出力では、メイン動画とマット動画を1組として扱います。どちらか一方でも既存ファイルと重複する場合、両方に同じ番号を付けます。

```text
song_spectrum.mp4
song_spectrum_matte_dark.mp4
```

のどちらかが既に存在する場合、次は次のようになります。

```text
song_spectrum (1).mp4
song_spectrum (1)_matte_dark.mp4
```

## 8. 長尺動画

30fps の長尺動画では生成に時間がかかります。60分動画ではフレーム数が 108,000 枚になります。生成中はログを確認し、必要であれば先に30秒プレビューで設定を確認してください。

## 9. 推奨設定

通常は以下から始めるのが安全です。

```text
表示タイプ：片側バー または 両側バー
色モード：縦グラデーション
バー色1：白
バー色2：白
背景色：黒（詳細設定、通常変更しない）
ペア出力：必要に応じてON
CRF：18
Encoder：libx264
```

## 10. トラブルシューティング

### ffmpeg が見つからない

`ffmpeg` と `ffprobe` をアプリ所在地の `bin` フォルダに置くか、PATH に入っているか確認してください。

### 動画編集ソフトでうまく重ならない

まずはペア出力を使い、マットを「比較（暗）」、メインを「比較（明）」で重ねてください。

### 色付きバーが思った通りに見えない

動画編集ソフトのブレンドモードによって見え方が変わります。まず白→白の単色で確認し、その後に色を付けるのが安全です。

### ffmpeg / ffprobe の参照順

アプリは、ffmpeg / ffprobe を次の順番で探します。

1. アプリ所在地の `bin` サブフォルダ
   - Windows例: `bin\ffmpeg.exe`
   - Windows例: `bin\ffprobe.exe`
2. PATH

EXE化した場合は、EXE と同じ場所に `bin` フォルダを作り、その中に `ffmpeg.exe` と `ffprobe.exe` を置く運用ができます。`bin` に存在しない場合は、従来どおり PATH を参照します。
